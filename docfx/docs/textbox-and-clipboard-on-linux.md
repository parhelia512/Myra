Myra uses [TextCopy](https://github.com/SimonCropp/TextCopy) for the text clipboard operations.

This library requires `clip` to be installed to work on Linux.

If it's not installed, Myra will fall back to its own clipboard, which will work only within the application using Myra (i.e., it won't be possible to select/copy text in another app and paste it into an app using Myra).
